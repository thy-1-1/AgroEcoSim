# Figure Catalog

This directory is reserved for generated result figures. The portfolio currently
uses curated SVG assets in `assets/` and `docs/assets/` so the GitHub README and
Pages experience remain lightweight and reliable.

Planned figure groups:

- species trajectories and phase portraits;
- scenario stability and chemical-pressure comparisons;
- PSR indicator dashboards;
- interaction-matrix heatmaps;
- uncertainty interval plots;
- data-assimilation state update curves.
